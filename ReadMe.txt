In order to run this code:
1. Pull code from Github
2. Open terminal
3. “cd” into file that has Main.java
4. enter the command: “javac Main.java” 
	|-> this will compile the java code
5. enter the command: “java Main” 
	|-> the program will execute
6. Optional: Update input.txt file
	|-> repeat step 4 (no need to recompile program)
