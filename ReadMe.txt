In order to run this code:
1. Pull/Clone code from Github 
2. enter the command: “javac Main.java” 
	|-> this will compile the java code
	|-> note user should already be "cd" into the directory that has Main.java, if not "cd" into the correct directory 
3. enter the command: “java Main” 
	|-> the program will execute
4. Optional: Update input.txt file
	|-> repeat step 4 (no need to recompile program)
